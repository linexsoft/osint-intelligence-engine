## What is it?

OSINT Intelligence Engine is a desktop application that accelerates open source intelligence research. Fill in the form, click generate — get 100+ ready-to-use search queries instantly, grouped by category.

---

## Features

- **👤 Person Investigation** — Dorks based on name, city, job, phone, email and social media handles
- **🏢 Company Intelligence** — Financial records, employee discovery, reputation and news queries
- **🌐 Domain / IP Recon** — Subdomain enumeration, technology fingerprinting, security vulnerability dorks
- **💥 Breach Detection** — Email, username, phone and hash-based leak hunting across paste sites and GitHub
- **🛠️ Built-in Tools** — Hash identifier, email validator, phone analyzer, username scanner
- **🔧 Custom Dork Builder** — Visually combine site:, filetype:, inurl:, intext:, intitle: operators
- **🔍 6 Search Engines** — Google, Bing, DuckDuckGo, Yandex, Brave, Startpage
- **📋 History & Export** — Export as HTML report, TXT or JSON
- **🌍 Turkish / English** — Full UI language support

## Usage

1. Launch the application
2. Select a category — Person / Company / Domain / Breach
3. Fill in the relevant fields
4. Click **▶ GENERATE**
5. Open queries individually or all at once, or export the results
